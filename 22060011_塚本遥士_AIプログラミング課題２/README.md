# 🐾 ランダム犬図鑑

## 概要
犬の画像をランダムに取得して表示するアプリです。「お気に入り」ボタンを押すと、画像URLと犬種をCSVファイルに保存できます。

## 使用API
- [The Dog API](https://dog.ceo/dog-api/)（APIキー不要）

## データベース機能
- `data/favorites.csv` に保存（加点対象）
